# hackfest
Conversational AI for instant business intelligence Dashboard
Interactive dashboard built with Streamlit.
Upload CSV files and generate charts automatically.
Supports bar charts, line charts, and pie charts.

Technologies Used:
  Python
  Pandas
  Streamlit
  Plotly

Run the program:
  install dependencies :
  pip install streamlit plotly pandas 

  run the dashboard :
  streamlit run app.py
